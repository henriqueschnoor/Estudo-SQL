# SELECT

O comando `SELECT` é utilizado para consultar dados em uma tabela.

## Exemplo básico

```sql
SELECT * 
FROM clientes;
```

Esse comando retorna todos os registros e todas as colunas da tabela `clientes`.

## Selecionando colunas específicas

```sql
SELECT nome, telefone
FROM clientes;
```

Esse comando retorna apenas as colunas `nome` e `telefone`.

## Uso no suporte

No suporte, o `SELECT` é usado para consultar informações de clientes, vendas, notas fiscais, produtos, usuários e registros do sistema.
