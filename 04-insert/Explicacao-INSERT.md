# INSERT

O comando `INSERT` é utilizado para inserir novos registros em uma tabela.

## Exemplo

```sql
INSERT INTO clientes (nome, telefone, cidade)
VALUES ('Henrique', '19999999999', 'Sao Paulo');
```

Esse comando cadastra um novo cliente.

## Uso no suporte

No suporte, o `INSERT` pode ser usado para criar registros de teste, cadastrar dados manualmente ou simular situações em ambiente de homologação.

> Atenção: em ambiente de produção, use com cuidado e sempre com autorização.
